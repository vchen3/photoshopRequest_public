Background

In this survey, you will rate ten posts from www.reddit.com/r/PhotoshopRequest, a website where users submit Photoshop jobs. You will receive an original image, a request for how it should be edited, and multiple response images that have been edited accordingly.

If you encounter a post with multiple source images, the "Original Image and Request" panel will show all images. Please compare the response image with the source image that matches it best. Some of the response images may look very similar.

Please complete the tasks described below for each of the following posts.

1) Note invalid requests

Imagine that you are an artist trying to edit the picture according to the request. A request is valid if you know exactly what edits to make based on the request and the photo. Invalid requests are non-specific instructions such as unclear requests or creative generation tasks. If you think the request is invalid, please click the "Invalid Request" button.

Example valid requests: "Remove the people in the background", "Brighten the color of the leaves."

Example invalid requests: "Put me somewhere awesome", "Please make me a logo for my project."

2) Note irrelevant responses

A response image is "irrelevant" if the image link is broken, or if it is a "troll" image: edited images that, while humorous and technically well-edited, are not what you think the user intended to receive. If you think the image is irrelevant, please click the "Irrelevant Response" button.

3) Rate image response

If image response is relevant, please give it a rating score. If this was your photo, how satisfied would you be with this edit on a scale from 1-5? Ratings should be independent of previous responses or requests.
