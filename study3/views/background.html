Instructions

In this survey, you will view ten pairs of images. Imagine that you have the original image but want to transform it into the edited image. For each pair, please type 1-2 sentences of instructions that you would give someone to make these edits. There is no wrong answer. Use the arrow keys to flip between the original and edited image to clearly view the changes. See example below.

"Replace the background with shadowy figures", or "Make me look like a businessman."

If you encounter a post with multiple original images, icons of each original image will be displayed. Please compare the edited image with the original image that matches it best, or if appropriate describe the edit by referencing multiple photos. See example below.

"Please add the guy on the far left of the first image into the second family photograph."
